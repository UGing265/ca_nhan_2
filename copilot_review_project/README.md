# Copilot Review Project

## Overview
An AI-powered code review tool with a VSCode-like UI built with Streamlit and Google ADK.

## Features
- VSCode-like code editor (Monaco Editor)
- AI-powered code review against custom policies
- Human-in-the-Loop repair decision workflow
- Stateful session management with Google ADK

## Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Configure `.env` with your GOOGLE_API_KEY
4. Run: `streamlit run streamlit_app.py`
